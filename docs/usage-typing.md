# 基本功能

本节的功能介绍以 Windows 版为例，Linux 和 Android 版本类似。
> 一些在 Windows 或 Linux 版本中需要特殊键位（如 Ctrl）引导触发的功能，在 Android 版本中可能因键盘上无对应按键、皮肤也未提供相应功能而不能使用。

## 选择方案

参见[安装小小星空](/install.md)章节关于“选择方案”的说明。

## 方案学习

* 看**入门**教程：本地教程文件位于[用户配置目录](/usage-custom#用户配置目录)的 doc 目录下。可通过键入命令直通车 `oorm` 或者输入法状态条右键菜单来快速访问此目录。
* 访问**官网**：键入命令直通车 `oogw`，或者通过输入法状态条右键菜单快速访问。
* **键盘**图：三种方法查看当前方案的键盘布局：①Ctrl+Shift+M；②键入命令直通车 `oojp`；③通过输入法状态条右键菜单。
* 反查编码：复制想查询编码的字词，然后 Ctrl+?。
* 联网查字：把想查询的字词打出到候选，然后 Alt+Enter。
* 笔画引导键：反引号 `` ` ``（位于数字`1`键左侧），例如 `` `ddh = 氵``。
* 万能键：`~`，例如 `bmov~~ = 辫`。
* 临时全拼引导键：`e`（仅用于星空两笔）。
* **单**字模式：Ctrl+Shift+D 切换，开启后不能打词，适合练习单字。

## 状态切换

* **简**繁切换：Ctrl+Shift+J
* 全半角切换：Ctrl+Space
* 中英文标点模式切换：Ctrl+.（或者按住 Alt 临时切换为英文标点模式）
* 临时英文模式：按左 Ctrl，则下次输入会变成临时英文模式。这是一种特殊的输入模式，在该模式下，你可以快速输入大写数字、日期、英文，甚至进行数学运算。注意：在临时英文模式下打英文，不能直接用数字键来上屏指定候选项，而要用 Alt+数字键。
![临时英文模式](/en-mode.png)

## 特殊字符

* 两分输入：在星空系列方案下，用 `u` 引导，如 `ulyly = 龖`
* 火星文（菊花文）模式：Ctrl+Shift+Alt+H 切换
* 软键盘：通过输入法状态条呼出。右击符号键盘，可切换符号类别
* **符**号大全：`of` 引导各类符号，`ol` 引导 LaTeX 公式，`ok` 引导假名（平文式罗马音），`op` 引导拼音和注音符号，`ot` 引导日期时间……按 `o` 即可看到提示
* 快捷符号：`;` 引导，可以快速输入少量常用符号

## 右键菜单

输入法状态条的右键菜单里集成了一些命令，它们也可以通过键入编码来执行：

* `ooxx` 提供与**小小**输入法程序相关的功能：重启小小、配置小小、重载小小、退出小小
* `oofk` **反馈**码表修改建议
* `ooml` 快速打开相关**目录**：程序目录、用户配置目录
* `oorm` 快速打开在线和本地**入门**说明书
* `oogw` 快速打开**官网**
* `oojp` 快速打开**键盘**图
