# 安装小小星空

小小输入法支持 Windows（XP-Win11）、Linux、Android（6.0+）系统，不支持 macOS 或 iOS 系统。

## Windows

### 安装

**方式一（自动安装）**：下载运行[小小星空输入法安装程序][小小星空网盘]。适合大多数情况。

::: info 选择一种安装模式
* 标准安装：安装过程会向系统注册 dll 文件，安装完成后输入法以 TSF 模式运行，具有更好的光标跟随能力。
* 便携安装：安装完成后输入法以外挂模式运行，可放 U 盘里随身携带。
:::

**方式二（手动安装）**：适合系统里已经安装了[原版小小输入法][小小输入法网盘]，并且不打算卸载它的情况。

1. 退出原版小小输入法。
2. 从[小小星空仓库][小小星空仓库]下载本项目源代码。
3. 将源代码中，home 目录里的内容（最关键的是 mb 和 entry 目录）复制到小小输入法的[用户配置目录](/usage-custom#用户配置目录)中。
4. 将 yong-win.ini 更名为 yong.ini（删掉原本的 yong.ini）。

自动或手动安装完成后，可以检查是否注册了 TSF 模块（即系统设置里是否添加了 Yong 输入法键盘），以判断输入法能否以 TSF 模式运行：

:::tabs
== Win11
![系统语言和键盘](/lang-and-kbd-win11.webp 'Win11：时间和语言 🞂 语言和区域 🞂 语言选项 🞂 键盘')
== Win10
![系统语言和键盘](/lang-and-kbd-win10.webp 'Win10：语言 🞂 中文语言选项 🞂 键盘')
== Win7
![系统语言和键盘](/lang-and-kbd-win7.webp 'Win7：区域和语言 🞂 键盘和语言 🞂 更改键盘')
== XP
![系统语言和键盘](/lang-and-kbd-xp.webp 'XP：区域和语言选项 🞂 语言 🞂 详细信息')

在 XP 系统上，还需要勾选“将高级文字服务支持应用于所有程序”，SP2 以下还需安装 wic 组件，才能以 TSF 模式运行输入法。
:::

### 启动

第一步（如果想要以外挂模式启动，请跳过此步）：把光标定位到任意文本框，然后如图在任务栏托盘或语言工具栏中切换到 Yong 输入法键盘（也可以用快捷键切换，一般是 Ctrl+Shift 或 Win+Space）。如果系统只有 Yong 输入法一个键盘，则不需要切换键盘，此步跳过。

:::tabs
== Win11
![切换系统键盘](/run-tsf-on-win11.webp 'Win10')
== Win10
![切换系统键盘](/run-tsf-on-win10.webp 'Win10')
== Win7
![切换系统键盘](/run-tsf-on-win7.webp 'Win7')
== XP
![切换系统键盘](/run-tsf-on-xp.webp 'XP')
:::

第二步：启动输入法主程序（可以从开始菜单启动，也可以运行程序目录下的 w64/yong.exe）。

第三步：呼出小小输入法的状态条。

:::tabs
== Win8 以上
把光标定位到任意文本框，即可使用 Ctrl+Space 快捷键，呼出（或隐藏）小小输入法的状态条。

如果成功安装了 TSF 模块，也可以鼠标点击任务栏托盘的「英」来呼出（或隐藏）小小输入法状态条。
== Win7 和 XP
使用 Ctrl+Space 快捷键，呼出（或隐藏）小小输入法的状态条。

如果成功安装了 TSF 模块，也可以鼠标点亮任务栏托盘的小小输入法图标，来呼出（或隐藏）小小输入法的状态条。
:::

::: tip 判断输入法当前运行模式的方法
右击输入法状态条，如果看到“TSF 模式未生效”的菜单项，说明正以外挂模式运行，否则以 TSF 模式运行。
:::

### 选择方案

尝试以下任意一种方式，打开 Yong 输入法设置程序：

* 点击输入法状态条上的齿轮图标
* 输入法状态条或托盘图标的右键菜单
* 键入命令直通车 `ooxx`

在设置程序中，加载你想使用的方案（如下图左）、卸载不想使用的方案（如下图右）。你可以放心地卸载内置的方案，因为是可以重新加载回来的。

![管理方案](/config-manage-schema.png '在 Yong 输入法设置程序中加载或卸载方案')

接着，尝试以下任意一种方式，切换到你想使用的方案：

* Ctrl+Shift+H
* 输入法状态条或托盘图标的右键菜单
* 点击输入法状态条上的方案名（但有些皮肤的状态条上不显示方案名）

### 退出 · 重启 · 重载

尝试以下任意一种方式：

* 小小输入法状态条的右键菜单
* 键入命令直通车 `ooxx`

::: info 重载和重启的区别
**重载**是指在不**重启**输入法的情况下，重新载入码表文件和大部分配置项。鼠标中键单击输入法状态条也可以触发重载。
:::

### 升级

安装到设备上的小小星空主要由两部分组成：小小输入法程序 + 星空系列汉字编码方案的码表。

* 升级方式一（升级程序和码表）：手动下载并运行最新的[小小星空输入法安装程序][小小星空网盘]。
* 升级方式二（只升级程序）：**右击输入法状态条 🞂 更新 🞂 更新程序文件**，可自动从小小输入法的服务器获取程序更新。
* 升级方式三（只升级键道方案码表）：**右击输入法状态条 🞂 更新 🞂 更新键道码表**，可自动将键道方案的码表同步至 [KeyTao][KeyTao官网] 最新状态。

::: info 为什么只能升级键道方案的码表？
因为其他方案暂时缺少积极的维护者。
:::

### 卸载

从开始菜单、控制面板或者程序目录执行卸载程序即可。

::: tip 在开始菜单里找不到卸载程序？
Win10 的一个[疑似 bug](https://answers.microsoft.com/zh-hans/windows/forum/all/%E7%A8%8B%E5%BA%8F%E7%9A%84%E5%8D%B8%E8%BD%BD/0b57356b-e988-486e-b863-1cf63e0cf0d2)导致开始菜单无法显示卸载程序，你可以改为运行程序目录下的 uninstall.exe。
:::

## Linux


::: info 并非通解
Linux 有很多发行版、桌面环境和输入法框架，本文只介绍少数场景下的安装方法，其他环境请参考[小小输入法官方文档](https://yong.dgod.net/read.php?tid=6)。
:::

### 安装

:::tabs
== Debian12 + GNOME + IBus
1. 下载[原版小小输入法 Linux 版][小小输入法网盘]（如 yong-lin-3.0.0.1.7z），解压后把里面的 yong 程序目录放到你喜欢的目录（如 /usr/local 或者 home 目录）。
2. 确保已安装了 im-config 包，然后进入 yong 程序目录，执行安装指令 sudo ./yong-tool.sh --install，注销并重新登录桌面。
3. 在**系统设置 🞂 键盘**里添加输入源，把中文 (Yong) 添加进去。
![添加输入源](/kbd-linux-gnome.webp '添加输入源')
4. 下载[本仓库][小小星空仓库]，将其中 home 目录里的内容（最关键的是 mb 目录和 entry 目录）复制到配置目录 ~/.yong（这是个隐藏目录），再把 yong-win.ini 重命名为 yong.ini（替换掉原本的 yong.ini，如果存在的话）。
> 如果你只想安装原版小小输入法（不含星空系列方案），可跳过此步。
5. 可选安装 gnome-shell-extension-appindicator 包（据说能改善托盘图标）。
== Debian12 + KDE Plasma (Wayland)
1. 下载[原版小小输入法 Linux 版][小小输入法网盘]（如 yong-lin-3.0.0.1.7z），解压后把里面的 yong 程序目录放到你喜欢的目录（如 /usr/local 或者 home 目录）。
2. KDE Plasma 默认安装了 fcitx5 框架，为了避免 Yong Wayland 受到干扰，要修改 ~/.profile，在底部添加以下环境变量设置：
```bash
export XMODIFIERS="@im=yong"
export GTK_IM_MODULE=wayland
unset QT_IM_MODULE
```
3. 对于 KDE Plasma 桌面环境，必须安装 libgtk-layer-shell0 包。此外，建议安装 libayatana-appindicator3-1（用于 gtk3 的 appindicator）包用于显示托盘图标，因为Wayland 下的输入法状态条几乎不可用，需要点击托盘图标才能打开输入法状态条的右键菜单。
4. 确保已安装了 im-config 包，然后进入 yong 程序目录，执行安装指令 sudo ./yong-tool.sh --install，注销并重新登录桌面。
5. 系统**设置 🞂 输入设备 🞂 虚拟键盘 🞂 无 🞂 Yong Wayland 🞂 应用**。
![添加输入源](/kbd-linux-kde.webp '切换虚拟键盘')
6. 下载[本仓库][小小星空仓库]，将其中 home 目录里的内容（最关键的是 mb 目录和 entry 目录）复制到配置目录 ~/.yong（这是个隐藏目录），再把 yong-win.ini 重命名为 yong.ini（替换掉原本的 yong.ini，如果存在的话）。
> 如果你只想安装原版小小输入法（不含星空系列方案），可跳过此步。
7. 修改 ~/.yong/yong.ini，在 [main] 节添加 wayland_show_hack=1。
:::

![管理方案](/config-manage-schema.png '在 Yong 输入法设置程序中加载或卸载方案')

接着，尝试以下任意一种方式，切换到你想使用的方案：

* 输入法状态条或托盘图标的右键菜单
* 点击输入法状态条上的方案名（但有些皮肤的状态条上不显示方案名）

### 退出

尝试以下任意一种方式：

* 小小输入法状态条或托盘图标的右键菜单

### 升级

安装到设备上的小小星空主要由两部分组成：小小输入法程序 + 星空系列汉字编码方案的码表。

前者可以通过以下任意一种方式更新：

* 输入法状态条或托盘图标的右键菜单
* 命令行执行 yong-config --update

后者目前没有自动升级方式，只能手动下载本项目的文件来进行升级。方法是：[退出](#退出)小小输入法，然后执行上方安装步骤的第 2 步。

### 卸载

进入程序目录，执行命令 sudo ./yong-tool.sh --uninstall。

## Android

### 安装

方式一（自动安装）：直接使用[小小星空输入法安装程序][小小星空网盘]。

方式二（手动安装）：

1. 安装[原版小小输入法 Android 版][小小输入法网盘]。
2. 下载[本项目源代码][小小星空仓库]，解压。
3. 将仓库的 home/yong-android.ini 更名为 yong.ini，复制并覆盖掉手机的 /sdcard/yong/yong.ini 和 /sdcard/yong/.yong/yong.ini。
> /sdcard/yong/.yong 是隐藏目录，你可能要在文件管理器中开启显示隐藏目录功能才能看到。
4. 将仓库的 home/version.txt 复制并覆盖掉手机的 /sdcard/yong/version.txt。
5. 将仓库的 home/skin-android 里的内容复制到手机的 /sdcard/yong/.yong/android 目录中。
6. 将仓库的 home 目录里的其他内容（最关键的是 mb 目录）复制到手机的 /sdcard/yong/.yong 隐藏目录中。

### 启动

如果你是初次安装，为确保正常使用，请执行以下步骤：

1. 确保 /sdcard/yong 目录不存在，如果存在请在文件管理器中删除或重命名之。
2. 运行主屏幕上的 Yong 输入法进入设置程序，授予有关权限，如：悬浮窗权限（必须的）、所有文件管理权限（必须的）、访问媒体文件、接收短信、麦克风权限、读取联系人、摄像头、读取剪贴板等。
3. 在手机系统设置的“管理输入法”（可通过 **Yong 输入法设置程序 🞂 安卓 🞂 设置系统输入法**快速打开）中，启用 Yong 输入法键盘。
4. 在任意文本框呼出当前的输入法键盘，并切换至 Yong 输入法键盘（如果当前输入法不提供切换功能，就在手机系统设置的“当前输入法”中切换），这时大概率不会弹出 Yong 输入法键盘，是正常的。
> 这步操作目的是在手机的 /sdcard/yong/.yong 隐藏目录下自动生成 yong.ini。作为替代手段，你也可以手动把 /sdcard/yong/yong.ini 复制为 /sdcard/yong/.yong/yong.ini。

不管是否初次安装，都继续执行以下步骤：

1. 在 Yong 输入法设置程序中进行个性化设置（例如在“输入法”页切换输入方案，在“安卓”页选择皮肤等；注意每个页面修改后要点右上角的保存才会生效）。稍等几秒后继续下一步。
![设置程序](/config-android.webp 'Yong 输入法 Android 版的设置程序')
2. 点进任意文本框弹出当前的输入法键盘，然后切换至 Yong 输入法键盘（如果当前的输入法键盘不提供切换键盘的功能，就在手机系统设置中修改“当前输入法”为 Yong 输入法，然后点进任意文本框）。如果未能成功弹出 Yong 输入法键盘，就在 Yong 输入法的“应用详情”中，强行停止并重新打开，然后重试此步。
![管理输入法](/config-android-current-ime.webp '当前输入法和管理输入法')

### 选择方案

运行主屏幕上的 Yong 输入法进入设置程序，在“输入法”页可切换输入方案。

### 退出

长按主屏幕上的 Yong 输入法图标，在“应用详情”中强行停止即可。

### 升级

安装到设备上的小小星空主要由两部分组成：小小输入法程序 + 星空系列汉字编码方案的码表。

* 升级方式一（升级程序和码表）：手动下载[小小星空输入法安装程序][小小星空网盘]并覆盖安装。
> 注意：如果新安装包的版本与已安装的相同，将不会覆盖已经存在的码表文件。
* 升级方式二（只升级程序）：运行主屏幕上的 Yong 输入法进入设置程序，在 **服务 🞂 更新**中下载最新的原版小小输入法安装包并覆盖安装。

### 卸载

像卸载普通 app 一样卸载后，还需要删除 /sdcard/yong 目录。
